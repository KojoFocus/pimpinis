# Step N Style - User Manual

Welcome to Step N Style! This guide will help you navigate and use our fashion e-commerce platform.

## For Customers

### Browsing Products
1. **Homepage**: View featured categories and latest products
2. **Shop Page**: Browse all products with search and category filters
3. **Category Pages**: Click category cards to view products in specific categories

### Shopping
1. **Search**: Use the search bar to find products by name or description
2. **Filter**: Click category buttons to filter products
3. **Product Details**: Click any product to view details, images, and add to cart
4. **Cart**: View cart contents, update quantities, or remove items
5. **Checkout**: Provide customer details to place orders

### Navigation
- **Home**: Featured categories and products
- **Shop**: All products with filters
- **Cart**: Shopping cart
- **Track**: Order tracking (if order ID available)

## For Administrators

### Dashboard Overview
- **Revenue Metrics**: Total revenue, monthly/weekly/daily breakdowns
- **Order Statistics**: Pending orders, total orders
- **Product Inventory**: Active products, low stock alerts

### Managing Products
1. **Add Product**:
   - Click "Add Product" from dashboard
   - Fill in product details (name, description, price, stock)
   - Upload product images
   - Select category
   - Set variants (sizes, colors if applicable)

2. **Edit Product**:
   - Go to Products section
   - Click edit icon on any product
   - Update details and save

3. **Delete Product**:
   - Use delete button in product list (with confirmation)

### Managing Categories
1. **Add Category**:
   - Go to Categories section
   - Click "Add Category"
   - Enter name (slug auto-generated)

2. **Edit Category**:
   - Click edit icon next to category
   - Update name and save

3. **Delete Category**:
   - Click trash icon (products will lose category association)

### Order Management
1. **View Orders**: See all orders with status, customer details
2. **Update Status**: Change order status (pending → confirmed → shipped → delivered)
3. **Track Revenue**: Monitor sales performance

## Features Guide

### Product Images
- Upload multiple images per product

### Search & Filters
- Real-time search across product names and descriptions

### Cart Functionality
- Persistent cart across sessions
- Quantity updates
